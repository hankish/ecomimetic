<p class="lead">The Standard Org Model is a reference architecture for ecomimetic organizations — a common language for structure, roles, and domains.</p>

## Purpose of the Model

The Standard Org Model does not prescribe a single organizational form. It provides a shared vocabulary that allows ecomimetic organizations to describe themselves, compare their structures, and learn from one another.

## Relationship to the Ecomimetic Org

The Standard Org Model is the specification; the Ecomimetic Organization is the living instance. No organization will map perfectly to the model, and that is expected. The model is a tool for reflection, not a template for replication.
