## What This Manual Is

This manual is not a recipe book. It does not tell you what to build or how to build it. Instead, it offers a set of principles, frameworks, and heuristics for developing your own judgment about how to design well within living systems.

Read it linearly the first time. Return to individual sections as specific questions arise in your practice. The manual is most useful when held alongside a real project — as a thinking partner, not an authority.
